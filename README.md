# 📊 Telco Customer Churn Analysis using Python

## 📌 Overview

This project demonstrates an end-to-end **Data Analytics** workflow using **Python**. The objective is to analyze Telco Customer Churn data by performing data loading, exploratory data analysis (EDA), data cleaning, and visualization to uncover patterns that influence customer churn.

The analysis focuses on understanding customer demographics, service subscriptions, contract types, payment methods, and tenure to identify key factors contributing to customer churn and support data-driven business decisions.

---

## 📂 Dataset

The project uses the **Telco Customer Churn** dataset containing customer information, including:

- Customer Demographics
- Customer Tenure
- Services Subscribed
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn Status

## 📈 Project Workflow

### 1. Data Loading
- Imported the CSV dataset into Python
- Loaded the dataset using Pandas
- Verified data types and dataset structure

### 2. Exploratory Data Analysis (EDA)
- Explored dataset dimensions
- Checked missing values
- Identified duplicate records
- Generated descriptive statistics
- Analyzed categorical and numerical variables

### 3. Data Cleaning
Performed data preprocessing by:

- Handling missing values
- Removing duplicate records
- Correcting data types
- Cleaning inconsistent values
- Preparing the dataset for analysis

### 4. Data Visualization

Created insightful visualizations using **Matplotlib** and **Seaborn**, including:

- Churn Distribution
- Customer Demographics Analysis
- Contract Type Analysis
- Monthly Charges Distribution
- Tenure Distribution
- Payment Method Analysis
- Correlation Heatmap
- Service-wise Churn Comparison

- Analyzed customer demographics and service usage

## ▶️ How to Run

1. Clone this repository

```bash
git clone https://github.com/RohitChxudhary/Telco-Customer-Churn-Analysis.git
```

2. Navigate to the project directory

```bash
cd Telco-Customer-Churn-Analysis
```

3. Install the required Python libraries

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open **Telco Customer Churn.ipynb**

6. Run the notebook cells sequentially to perform data loading, cleaning, EDA, and visualization.
